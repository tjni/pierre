# Pierre Theme

A clean light and dark theme for Visual Studio Code and Cursor, built for
[Diffs.com](https://diffs.com).

## Installation

1. Open the Extensions view (`Cmd+Shift+X`)
2. Search for **Pierre Theme**
3. Click **Install**
4. Open the Command Palette (`Cmd+Shift+P`) → **Color Theme** → select **Pierre
   Light** or **Pierre Dark**

## Variants

- Pierre Light
- Pierre Dark
- Pierre Light Soft
- Pierre Dark Soft
- Pierre Light Protanopia & Deuteranopia
- Pierre Dark Protanopia & Deuteranopia
- Pierre Light Tritanopia
- Pierre Dark Tritanopia
- **For web-use only:**
  - Pierre Light Vibrant
  - Pierre Dark Vibrant
